# Store
Aplicação para cadastro de usuários e seus pedidos. 

##v1
* ✅ API Rest: Gin-Gonic;
* ✅ ORM: GORM;
* ✅ Testes: Gomock; e
* ✅ Persistência: MySQL.

##v2
* ✅ API Rest: Gin-Gonic;
* ✅ ORM: GORM;
* ✅ Testes: Gomock **para UseCases e Handlers**;
* ✅ Persistência: MySQL e **Elastic Search**; e
* ✅ **Docker**.

##v3
* ✅ API Rest: Gin-Gonic;
* ✅ ORM: GORM;
* ✅ Testes: Gomock para UseCases e Handlers;
* ✅ Docker;
* Persistência: MySQL **com** Elastic Search; e
* Cache: **Redis**.
